transform_in_linktransform from the link frame to the sensor frame
filter_cutoff_frequencycutoff frequency of the filter (in Hz). If 0, the filter is not enabled

Parameters

in_sensor_frameif true, the force is expressed in the sensor frame, if false, it is in world frame
Returns
const Eigen::Vector3d& The force sensed by the simulated force sensor
