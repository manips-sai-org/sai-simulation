Detailed Description

Class to represent a simulated force sensor. note that this implementation ignores the mass and inertia of the object attached beyond the force sensor. note also that this assumes the force sensor to be located just between the link and the end-effector. That is, it cannot account for any joint reaction forces in case the sensor is to be attached on the link between two joints.
